# Video Streamming

Este projeto é uma plataforma de _streaming_ de vídeo em desenvolvimento, criada como um laboratório para aplicar e evoluir conceitos de arquitetura modular e arquitetura evolutiva.

**Objetivo**

- Implementar uma solução de streaming de vídeo organizada em módulos claros, com responsabilidades bem definidas.
- Usar práticas de arquitetura evolutiva para que o sistema possa crescer, ser refatorado e adaptar-se conforme novas necessidades surgem.

**Arquitetura**

- Arquitetura modular: cada domínio ou camada (ex.: upload, processamento, armazenamento, entrega, API) é implementado como um módulo independente, com interfaces bem definidas.
- Arquitetura evolutiva: mudanças e melhorias devem ser introduzidas em pequenos passos, com feedback contínuo — priorizando testes, observabilidade e refatoração incremental.

**Branches como passos da evolução**

Cada branch criada neste repositório representa um passo na evolução do projeto. A ideia é que cada branch contenha uma melhoria, experimento ou refatoração significativa, permitindo acompanhar o histórico de decisões arquiteturais.

Esse repositório é um espaço de aprendizagem e evolução contínua: trate cada branch como um passo deliberado na construção de uma plataforma de streaming mais robusta e modular.
